# Examples: the kit applied to real public articles

Field notes from running the [rubric](../content-readiness-rubric.md) and [compression test](../compression-test-template.md) on real, public articles. The point is to make the method concrete - what an AI extracts, what survives compression, and where the lift is.

**A note on fairness.** These are public articles analyzed to demonstrate the kit, not to grade the authors. Strong content is celebrated as a model; gaps are shown only alongside the fix. Every source is linked - go read the originals. The compression verdicts are honest single-pass reads by one model; a different model or prompt will word them differently, but the direction (what survives, what averages out) is stable.
